CSSBlock.jsとは

CSSBlock.jsとは、javascriptでCSSをブロックに分けて扱うことを目的とした軽量なライブラリです。CSSBlock.jsを使えば、セレクタ部分のみを扱ったり、プロパティのみを取り出したりできます。

例

例えばCSSBlockに以下のようなコードを渡したとします。 

CSS

.button{
	color:#000;
	border:1px solid #999;
	border-radius:5px;
}
 
Javascript

CSSBlock.init(code);

すると、セレクタ部分とプロパティ部分は次のようになります。 

cssBlock.eq(0).selector;// ".button"
cssBlock.eq(0).properties;
/*
	color:#000;
	border:1px solid #999;
	border-radius:5px;
*/
