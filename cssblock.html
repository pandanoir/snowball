CSSBlock.jsとは

CSSBlock.jsとは、javascriptでCSSをブロックに分けて扱うことを目的とした軽量なライブラリです。CSSBlock.jsを使えば、セレクタ部分のみを扱ったり、プロパティのみを取り出したりできます。

例

例えばCSSBlockに以下のようなCSSソースを渡したとします。 

CSS
.button{
	color:#000;
	border:1px solid #999;
	border-radius:5px;
}
 
Javascript
var css=new cssBlock(code);

すると、渡したCSSソースを分解します。分解すると、次のようになります。 

Javascript
css.parseType();//[[".button","OTHERS"],["{","LEFT_CURLY_BRACKET"],["\n","NEWLINE"],["\t","TAB"],["color","PROPERTY"],
[":","COLON"],["#000","BLOCKS"],[";","SEMICOLON"],["\n","NEWLINE"],["\t","TAB"],
["border","PROPERTY"],[":","COLON"],["1px solid #999","BLOCKS"],[";","SEMICOLON"],["\n","NEWLINE"],
["\t","TAB"],["border-radius","PROPERTY"],[":","COLON"],["5px","BLOCKS"],[";","SEMICOLON"],
["\n","NEWLINE"],["}","RIGHT_CURLY_BRACKET"]]

cssBlockには以下のメソッドがあります。

push()
cssBlockの最後尾に要素を追加します
noConflict()
cssBlockを無効化し、もしこのライブラリより前にcssBlockが定義されていたらそれに置き換えます。
getBlock()
要素の一覧を取得します。
joinBlock()
要素をつなげて文字列として返します
selector()
指定した属性のものを返します
setBlock()
要素を更新します
deleteBlock()
要素を削除します
insertBlock()
要素を指定箇所に挿入します
getOriginal()
cssBlockでパースしたコードを返します
parseType()
joinBlock()とほぼ同じですが、属性を数値ではなく文字列に変換して返します
